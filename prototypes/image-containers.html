Image Containers

Reference

Image Aspect Ratio Key

All images render inside a fixed 3:2 container. Standard 3:2 photos use object-fit: cover to fill the frame edge-to-edge. Non-standard aspect ratios — vertical and panoramic — use object-fit: contain so no portion of the image is cropped; the secondary background color fills any empty pillarbox or letterbox area. This approach keeps grid layouts stable regardless of what the photo desk delivers.

Lead Story Images

Ratio — 3 : 2 Standard Landscape

object-fit: cover fills the 3:2 frame edge-to-edge. No empty space, no layout shift.

Ratio — Taller than 3 : 2 Vertical / Portrait

object-fit: contain within the 3:2 frame. Pillarbox areas use a white background so they blend into the page.

Ratio — Wider than 3 : 2 Panoramic / Wide

No fixed container. Height follows the image's natural dimensions — no letterbox fill needed.

Grid / Story Card Images

Ratio — 3 : 2 Standard Landscape

object-fit: cover fills the 3:2 frame edge-to-edge. No empty space, no layout shift.

Ratio — Taller than 3 : 2 Vertical / Portrait

object-fit: contain within the 3:2 frame. Pillarbox areas filled with the secondary background to preserve grid alignment.

Ratio — Wider than 3 : 2 Panoramic / Wide

object-fit: contain within the 3:2 frame. Letterbox areas filled with the secondary background to preserve grid alignment.
